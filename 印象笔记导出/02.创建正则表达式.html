02.创建正则表达式

使用RegExp构造函数，

var str = "fuck  ni zhen  de fuck fuck Fuck";
        // 一、创建正则表达式
        // 1.字面量 var reg = /字符串或者匹配规则/gi
        var reg = /fuck/gi;
        str.replace(reg,"**");
        // 2.构造函数 var reg = new RegExp("字符串","gi")
        //    *var reg = new RegExp(变量,"gi")
        var reg = new RegExp("fuck","gi");
        str.replace(reg,"**")
  • 第一个参数就是我们的模式“字符串”
    var reg= new RegExp('study');

    //使用特殊字符
    var reg= new RegExp('\\d\\w+');\d\w+
  • 第二个参数可选，模式修饰符
    • i: case-insensitive，表示忽略大小写
    • g: global，表示全局匹配
    • m: multiline，表示多行匹配
  • var reg = new RegExp('study', 'ig');//变量用引号，匹配也用引号

还可以用字面量方式直接声明

var reg = /study/gi;
  • 直接量是字符匹配，不支持变量，也不用引号
