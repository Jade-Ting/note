09.函数的递归

函数递归

函数可以自己调用自己, 成为函数的递归调用（函数自己执行自己）
  • 需要一个退出函数的条件（return），若不存在退出条件会出现死循环
    • 备注：报错Maximum call stack size exceeded 表示超出最大内存（死循环）
  • 递归前输出，正向输出
  • 递归后输出，按最后一次递归的值输出
  // (二)递归:函数自己执行自己
        // * 需要一个退出函数return的条件，若不存在退出条件会死循环。
        var num = 10;
        // function zijian(){
        //     num--;
        //     if(num <= 0){return;}
        //     console.log(num);//递归前输出，正向输出
        //     zijian();
        // }
        
        function zijian(){
            num--;
            if(num <= 7){return;}
            zijian();
            console.log(num);//递归后输出，按最后一次递归的值输出
        }


        zijian();
        // 报错：Maximum call stack size exceeded 超出最大的内存

递归调用的过程:
  1. 首先去找临界值，即无需计算，获得的值(一般是返回该值)。
  2. 找这一次和上一次的关系(一般从后往前找)
  3. 假设当前函数已经可以使用，调用自身计算上一次的运行结果，再写出这次的运行结果。
