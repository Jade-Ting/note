06.数组的复制与传递

基本数据类型的传递
    var a = 10;
    var b = a;
    var b = "haha";
    console.log(a,b);
            ---->a:10,b:haha
原数据的值不会被改变。
引用数据类型的传递（数组的传递）
    var arr = [10,20,30];
    var brr = arr;
        brr[0]  = "haha" ;
        console.log(arr,brr);
                ------>arr[haha,20,30]
                            brr[haha,20,30]
传递的是地址，原数组的值会跟着被改变。

数组复制与传输

如何复制数组（且不影响到原数组）

  • 遍历后通过push（）把原数组的元素复制到新数组中。
    var res = [];
    for(){res.push(arr[i])}
  • 通过slice(0)将原数组的元素切割后再push到新数组中。使用slice原数组不受影响。
        var res1 = arr1.slice(0);
        res1.push("laoxie");

        console.log("arr1",arr1);
        console.log("res1",res1);
