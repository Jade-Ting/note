02.数组操作

(1)数组的读取（查）与写入（赋值）

* 数组的索引 arr[index]

index代表索引值，从0开始计数。
var arr = ['html5','css3','javascript'];

//读取
arr[0]; //=> 'html5'
arr[2]; //=> 'javascript'

//写入
arr[3] = 'web前端';

(2)数组的遍历

* 数组的长度 arr.length
var arr = ['html5','css3','javascript'];
for(var i=0;i<arr.length;i++){
console.log(arr[i]);
}
