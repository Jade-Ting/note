### 添加指定文件夹

打开需要提交的指定文件夹的终端，正常执行添加执行的命令即可只提交这个文件夹。


### 添加指定文件

1. 添加指定文件到暂存区
```
git add 文件相对路径(在vscode中可右键获取文件相对路径)
```

2. 查看仓库状态(可忽略)
```
git status
```

3. 忽略其他文件，把现修改的隐藏起来，这样提交的时候就不会提交未被add的文件

(没有特殊情况时可忽略)
```
git stash -u -k
```

4. 从暂存区提交到本地版本库
```
git commit -m ''
```

5. 拉取合并(若远程没有做任何修改，可忽略)
```
git pull
```

6. 推送到远程仓库
```
git push
```

7. 恢复之前隐藏忽略的文件
```
git stash pop
```
